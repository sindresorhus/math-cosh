# math-cosh

> ES2015 [`Math.cosh()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/cosh) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install math-cosh
```


## Usage

```js
const mathCosh = require('math-cosh');

mathCosh(1);
//=> 1.5430806348152437
```
